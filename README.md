# Запуск приложения

```bash
# Перейти в папку с приложением и запустить
cd f2f-bank-tests
docker-compose up -d
# Приложение будет доступно на http://localhost
```

# Запуск тестов

```bash
# Установка зависимостей (один раз)
cd f2f-bank-tests
npm install

# Запуск всех тестов
npx playwright test

# Запуск конкретного файла
npx playwright test tests/auth/register.spec.ts

# Запуск конкретного теста по названию
npx playwright test --grep "TC:01"

# Просмотр отчёта
npx playwright show-report
```
